# Increasing

Description
------------

That dot seems familiar.

a
     .
   k
6


Key
---------
15611

Internal Description
--------------------
Translate all the symbols to numbers:

a -) 1
k -) 11
6 -) 6
. -) e (Morse code) -) 5

Then, put the number in increasing order to create a five-digit key:

1 < 5 < 6 < 11
15611
